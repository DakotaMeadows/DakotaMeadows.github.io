CSS Positioning

September 6, 2014


In CSS there are three properties for positioning where you want your elements on the page. They are fixed, absolute, and relative. If you put in "relative," the computer will then take whatever other directions you give it and change the position of the element starting from where it normally would be without any CSS properties. Typing in "absolute" will remove the element from the normal flow of the page (all other elements will ignore it). You can use this to make it overlap other elements and place the element anywhere without changing the properties of other elements. Using "fixed" will make the element stick to wherever part of the page you put it on. Scrolling will not change where it is, so you will see it no matter where you are on the page.
