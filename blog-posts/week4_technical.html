Enumerable Methods

September 21, 2014


First things first: Enumerable methods are commands that can be called on groups of objects, like a hash or an array. These are very useful because of going through the entire group and changing them/writing code for every single one, you can call an enumerable method, code for whatever you want to happen to every one, and then the computer does all the hard work for you. One example is the enumerable#map method. This is not a destructive method (does not overwrite the original array and permanently change it). For that, just add an exclamation point (called a "bang"). What .map does is take your code, and apply it to every element in an array. For example, if I have an array called "numbers" and it equals [1,2,3,4] and I want a new array that is equal to numbers times 5, I simple write numbers.map{|n| n*5}. This will make the new array equal to [5,10,15,20]. As you can see, this method severely cuts down the amount of time needed to alter an array if you want all the elements changed the same way.
