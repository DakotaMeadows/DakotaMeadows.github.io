Variable Scope

October 5, 2014


Variable Scope refers to where in a program the variable is accessible. Ruby has four types of variables (local, global, instance, and class) as well as a constant type of variable. In addition, there are two psuedo-variables which can't be assigned values (nil and self, which refers to the object currently being executed). Most of the time, just looking at the variable/code will tell you the scope of the variable, but if not, typing in "defined? variable" will tell you. for example: 

						@x = 20
						=> 20

						defined? @x
						=> "instance-variable"
